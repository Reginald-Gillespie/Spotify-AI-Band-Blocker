# AI Band Blocker

This plugin will automatically skip any known-AI artists. 

Install from the [Spicetify](https://spicetify.app/) Marketplace, or directly by copying the script from `./dist/ai_band_blocker.js`.

If you encounter unskipped AI artists, please file an issue in [this repo](https://github.com/romiem/ai-bands).
